# DarkMagic-UltraRapidFire

This a mod for Mirage Arcane Warfare which reduces the cooldown of every ability by 75%.

## How to setup

1 - The mod is server side, so download a server if you don't already have one.

2 - In your server files go to Latte Dedicated Server\TBL, and copy the Content folder that's in this git. It should only replace the modified UAssets.

3 - Configure and run your server normally.
